1秒未満を実現した5つの工夫

1. Gemma4 シンキングモード OFF

Modelfileで /think タグを無効化し、推論オーバーヘッドを除去

TEMPLATE """{{ if .System }}<start_of_turn>user
{{ .System }}<end_of_turn>
{{ end }}..."""

2. Ollama直結ストリーミング

Lambda / API Gatewayを経由せずブラウザから直接Ollamaに接続

fetch('http://localhost:11434/api/generate', {
  method: 'POST',
  body: JSON.stringify({ stream: true, ... })
})

3. 句読点ベースのチャンク分割

最初の句読点（。、！？）到達時点でTTS生成を即開始

if (PUNCTUATION_PATTERN.test(char) && buffer.length >= MIN_LEN) {
  yield buffer.trim(); // → TTS へ
}

4. TTS並行生成・Fire-and-Forget

TTS APIコールをawaitせずPromiseとして投げ、LLMストリームをブロックしない

synthesizeSpeech(chunk)          // awaitしない
  .then(buf => audioQueue.enqueue(buf))
  .catch(console.error);

5. クラウドTTSへの切り替え

ローカルCPU処理（TTFB 2秒〜）からAIVIS Speechクラウド（TTFB 250ms）へ移行

POST https://api.aivis-speech.com/v1/synthesis/stream
{ "text": chunk, "streaming": true }
