# DBに入れるデータ 全まとめ（西洋星占術）

---

## テーブル構成

```
zodiac_signs        → 12星座の基本データ
planets             → 10天体の意味・影響
houses              → 12ハウスの意味
aspects             → アスペクトの種類と意味
vocabulary          → カテゴリ別占い用語・フレーズ
advice_patterns     → 相談カテゴリ別アドバイス方向性
response_phrases    → 話し方サンプル（口調・表現）
conversation_nodes  → 会話フロー定義（分岐ノード）
```

---

## 1. zodiac_signs（12星座）

| フィールド | 内容 |
|---|---|
| id | 1〜12 |
| name_ja | 牡羊座〜魚座 |
| name_en | Aries〜Pisces |
| period | 例: 3/21〜4/19 |
| element | 火・地・風・水 |
| quality | 活動・固定・柔軟 |
| ruling_planet | 支配星（例: 火星） |
| keywords | 性格キーワード配列 |
| traits | 詳細な性格説明 |
| strengths | 長所 |
| weaknesses | 短所 |
| love_style | 恋愛スタイル |
| work_style | 仕事スタイル |

**データ例（牡羊座）:**
```json
{
  "name_ja": "牡羊座",
  "element": "火",
  "ruling_planet": "火星",
  "keywords": ["情熱", "行動力", "開拓者", "直感"],
  "traits": "エネルギッシュで行動力があり、新しいことへの挑戦を好む",
  "love_style": "積極的で情熱的。一目惚れが多い",
  "work_style": "リーダーシップを発揮。スピード重視"
}
```

## 6. advice_patterns（アドバイス方向性）

| パターン | 使う状況 | 例文 |
|---|---|---|
| 背中を押す | 木星・金星が好配置 | 「今こそ動くときです。星があなたを後押ししています」 |
| 慎重に待つ | 土星・逆行期 | 「今は種を蒔く時。焦らず機を待つことで実りが来ます」 |
| 変化を促す | 冥王星・天王星の影響 | 「変化を恐れないで。この転機はあなたの魂が求めているもの」 |
| 現状維持 | 安定した配置 | 「今の流れを大切に。無理に変えず、育てていきましょう」 |
| 内省を促す | 12ハウス・海王星 | 「内なる声に耳を傾けて。答えはあなたの中にあります」 |

## 7. response_phrases（話し方サンプル）

### 開始フレーズ
```
「星の導きのもと、あなたのご相談をお聞きします」
「宇宙はあなたのことを見ています。何をお知りになりたいですか？」
「今日の星の配置は、大切なことを教えてくれています」
```

### 共感フレーズ
```
「それは、星も感じているようです」
「その迷いは、あなたの魂が成長しようとしているサインです」
「宇宙はあなたの思いをしっかり受け取っています」
```

### 締めフレーズ
```
「星の光があなたの道を照らしてくれるでしょう」
「あなたには、その力が必ずあります」
「宇宙はあなたの味方です。信じて進んでください」
```
