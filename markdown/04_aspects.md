## メジャーアスペクト

### ☌ コンジャンクション（合）

| 項目 | 内容 |
|------|------|
| 角度 | 0° |
| 標準オーブ | ±8° |
| 性質 | 中性（関係する天体による） |

**キーワード：** 融合, 強烈さ, 合一, 混合, 集中, 新しい始まり

**解説：** コンジャンクション（合）は2つの天体が同じ度数（0度）に位置するアスペクト。両天体のエネルギーが融合し、強烈に表れる。関係する天体の性質によって吉凶が異なる。太陽と月のコンジャンクションは新月を示し、新しいサイクルの始まりを意味する。

**解釈：** 2天体のエネルギーが一体化し、その分野が人生の重要なテーマとなる。天体の性質が強化され、集中的に表れる。

### ⚹ セクスタイル（六分）

| 項目 | 内容 |
|------|------|
| 角度 | 60° |
| 標準オーブ | ±6° |
| 性質 | 調和的 |

**キーワード：** 機会, 協力, 才能, 容易さ, コミュニケーション, サポート

**解説：** セクスタイル（六分）は2つの天体が60度の角度を形成するアスペクト。調和的で好機をもたらす。トラインより弱いが、積極的に活用すれば大きな恩恵を得られる。異なる元素（火と風、地と水）の星座間で形成されることが多い。

**解釈：** 才能・機会・協力のエネルギー。努力することで活かせる好機。コミュニケーションと協力が鍵。

### □ スクエア（四分）

| 項目 | 内容 |
|------|------|
| 角度 | 90° |
| 標準オーブ | ±8° |
| 性質 | 困難・緊張 |

**キーワード：** 緊張, 対立, 挑戦, 摩擦, 行動, 苦闘を通じた成長

**解説：** スクエア（四分）は2つの天体が90度の角度を形成するアスペクト。緊張・対立・挑戦をもたらすが、その摩擦が成長の原動力となる。同じモダリティ（活動・不動・柔軟）の異なる元素の星座間で形成される。困難なアスペクトだが、克服することで大きな力となる。

**解釈：** 緊張と対立のエネルギー。2天体のエネルギーが衝突し、課題や摩擦が生じる。しかし、この緊張が行動と成長の原動力となる。

| 性質 | 調和的・恵み |

**キーワード：** 調和, 流れ, 生まれつきの才能, 容易さ, 優雅さ, 恵み, 創造性

**解説：** トライン（三分）は2つの天体が120度の角度を形成するアスペクト。最も調和的で恵みをもたらすアスペクト。同じ元素（火・地・風・水）の星座間で形成される。生まれつきの才能・自然な流れ・恵みを示す。ただし、容易すぎるため怠惰になる危険もある。

**解釈：** 調和と才能のエネルギー。2天体のエネルギーが自然に流れ、生まれつきの才能や恵みをもたらす。最も吉祥なアスペクト。

### ☍ オポジション（衝）

| 項目 | 内容 |
|------|------|
| 角度 | 180° |
| 標準オーブ | ±8° |
| 性質 | 困難・対立 |

**キーワード：** 対立, 二極性, 気づき, 人間関係, バランス, 投影

**解説：** オポジション（衝）は2つの天体が180度の角度を形成するアスペクト。対立・二極性・バランスを示す。対向する星座間で形成される。自分の中の対立する力・他者との対立・投影のパターンを示す。バランスを取ることで統合と成長が生まれる。

**解釈：** 対立と二極性のエネルギー。2天体のエネルギーが引っ張り合い、バランスを求める。他者との関係を通じて気づきを得る。

## マイナーアスペクト

### セミセクスタイル（三十分）（30°）

**性質：** 穏やかな調和 | **オーブ：** ±2°

**解説：** セミセクスタイル（三十分）は2つの天体が30度の角度を形成するマイナーアスペクト。穏やかな調和と微妙なつながりを示す。隣接する星座間で形成される。

**解説：** クインカンクス（百五十分）は2つの天体が150度の角度を形成するアスペクト。ぎこちなく調整困難な関係を示す。2つの天体は元素もモダリティも共有しないため、自然な調和が難しい。健康問題や方向転換を示すことが多い。
