## ディグニティの種類

| 品位 | 説明 |
|------|------|
| **ドミサイル（本来の家）** | 惑星が最も自然に機能する星座。エネルギーが純粋に表れる |
| **エグザルテーション（高揚）** | 惑星が特別に強く機能する星座。ドミサイルに次ぐ強さ |
| **デトリメント（不利）** | ドミサイルの対向。惑星の性質が不適切に表れやすい |
| **フォール（失墜）** | エグザルテーションの対向。惑星が最も弱く機能する |
